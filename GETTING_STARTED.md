# Getting Started with Edutime

This guide helps you set up **Edutime** locally and run it for the first time.

## Prerequisites

- **Flutter** 3.38+ (`flutter --version`)
- **Dart** 3.10+ (comes with Flutter)
- **Git**
- Android Studio / VS Code + Flutter & Dart extensions
- Supabase account (free tier is enough for development)
- Firebase project (for push notifications)

## Step-by-Step Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-org/edutime.git
